# Python---Logic--Building-Task--1
Logic Building Task - 1 is a beginner-friendly Python practice project that focuses on improving basic programming and problem-solving skills.
This project is a beginner-friendly Python logic building practice set created to strengthen fundamental programming concepts such as conditions, loops, variables, and basic problem solving.

It contains 5 small tasks implemented using pure Python (no external libraries required).

The tasks included in this project are:

User Login Check

Even or Odd Number Check

Find Largest of Three Numbers

Student Grade System

Error Message Detector

These tasks and sample outputs are based on the provided notebook content.
